# Trinity Academy Website - Features Guide

Welcome to the feature exploration guide for the new Trinity Academy website. This document highlights the key sections and functionalities available for review.

## 1. Core Pages & Navigation
- **Home**: Dynamic landing page featuring a premium Hero section, "Admissions Open" banner, and school highlights.
- **About Us**:
  - **Principal's Message**: A dedicated message to students and parents.
  - **Management / Trust**: Overview of the school's leadership team.
- **Academics**: Details about the curriculum, teaching methodology, and academic standards.
- **Our Team**:
  - **Advisory Board**: Modern glassmorphism profiles of school visionaries.
  - **Our Teachers**: Interactive faculty directory with **department-wise filtering** (Science, Math, etc.).

## 2. Interactive Features
- **Online Admission Form**: A modern, 4-step guided application process for new students.
- **Career Portal**:
  - Browse current job vacancies.
  - **Job Application Form**: Multi-step application with personal info, qualifications, and CV upload UI.
- **Dynamic Navbar**: Responsive navigation with premium dropdowns and a mobile-optimized menu.

## 3. Specialized Sections
- **Facilities**: Grid layout showcasing campus amenities (Library, Labs, Sports, etc.).
- **Gallery**: Visual showcase of school events and infrastructure.
- **News & Events**: Stay updated with the latest happenings at the school.
- **Downloads**: Quick access to important school documents and PDF forms.
- **Contact**: Integrated contact details, school address, and an inquiry map placeholder.

## 4. Design & Technology Highlights
- **Premium Aesthetics**: Utilization of glassmorphism, vibrant mesh gradients, and "Inter/Outfit" typography.
- **Smooth Animations**: Powered by `framer-motion` for staggered entrances and micro-interactions.
- **Mobile First**: Fully responsive design that works seamlessly on smartphones and tablets.
- **High Performance**: Built with Vite and React for lightning-fast page transitions.

---

### How to Explore
1. Start at the [Home Page](http://localhost:5173/).
2. Experiment with the **"Our Team"** dropdown to see the filtering in action on the Teachers page.
3. Try clicking **"Apply Now"** in the Careers section to see the new application flow.
4. Fill out a sample **Admission Form** to experience the guided progress tracking.
